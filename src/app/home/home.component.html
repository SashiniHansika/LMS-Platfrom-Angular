Course Description

Introduction:

This course is aimed at advancing concepts of C programming and software code organization within the framework of structural and procedural programming paradigms.
This e-Learning course is organized as a series of lessons with different types of lesson materials as well as practice quizzes, assignments, discussion forums etc.

Target audience:

Any student who would love to learn C programming.

Course objectives:

  • Write a basic C program.

  • Compare different loops and the situation which that loops can use.

  • Illustrate the structure of the programme for given problem.

  • Explain about different data structures which can be used for different problems.

  • Recommend different efficiency programmes for a given problem.

Instructions for the learners:

  • Follow lessons in order.

  • You can read either lecture material or watch the video of the lesson.

  • Take the quizzes and evaluate your knowledge.

  • Make sure to do the Assignemnts on time.

  • Actively particiapte to the discussion forums.
